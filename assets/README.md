# SNAK — Assets

This directory contains game assets.

## Fonts
The game uses **Orbitron** and **Rajdhani** fonts (Google Fonts).

Download from:
- https://fonts.google.com/specimen/Orbitron
- https://fonts.google.com/specimen/Rajdhani

Place font files in `assets/fonts/`:
- `Orbitron-Regular.ttf`
- `Orbitron-Bold.ttf`
- `Rajdhani-Regular.ttf`
- `Rajdhani-Bold.ttf`

## Sounds (optional)
Place sound effect files in `assets/sounds/`:
- `eat.mp3` — food eaten sound
- `powerup.mp3` — power-up collected
- `gameover.mp3` — game over sound
- `background.mp3` — background music loop

## Images
Place image files in `assets/images/`:
- `icon.png` — app icon (1024×1024)
